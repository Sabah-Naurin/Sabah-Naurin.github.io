Datatypes & Variables

Data Types in Java

Java is a statically-typed language, meaning you must declare the data type of a variable when you create it. Java has two main categories of data types:

a) Primitive Data Types

These are the most basic data types in Java, and they store simple values like numbers, characters, and true/false values.

Data Type Size Description Example
byte 1 byte Small whole numbers (-128 to 127) byte age = 25;
short 2 bytes Whole numbers (-32,768 to 32,767) short population = 15000;
int 4 bytes Whole numbers (-2 billion to 2 billion) int salary = 50000;
long 8 bytes Large whole numbers long distance = 100000L;
float 4 bytes Decimal numbers (7 decimal digits) float pi = 3.14f;
double 8 bytes Decimal numbers (15 decimal digits) double gpa = 3.75;
char 2 bytes Single character (letter, digit, symbol) char grade = 'A';
boolean 1 bit True or false values boolean isJavaFun = true;

b) Non-Primitive or Reference Data Types

These include strings, arrays, and objects. Non-primitive types are more complex and built from primitives. Reference variables donâ€™t store the actual data but store the memory address (reference) of the object they point to.

Declaring Variables

In Java, you declare a variable by specifying its type, followed by its name. You can also assign a value to it when you declare it.

dataType variableName = value;

Example:


int age = 20;        // Integer variable
double salary = 4500.50; // Decimal number
char grade = 'A';    // Single character
boolean isJavaFun = true; // True/False value
String name = "Alice"; // Text

Examples of Using Variables

Example 1: Declaring and Printing Variables


public class VariablesExample {
  public static void main(String[] args) {
      int age = 25;
      double gpa = 3.8;
      char grade = 'A';
      String name = "John";
      boolean isGraduated = false;

      System.out.println("Name: " + name);
      System.out.println("Age: " + age);
      System.out.println("GPA: " + gpa);
      System.out.println("Grade: " + grade);
      System.out.println("Graduated: " + isGraduated);
  }
}

Examples of Using Variables

            
Example 2: Arithmetic Operations with Variables

            

public class ArithmeticExample {
  public static void main(String[] args) {
      int num1 = 10;
      int num2 = 5;

      System.out.println("Sum: " + (num1 + num2));          // Addition
      System.out.println("Difference: " + (num1 - num2));   // Subtraction
      System.out.println("Product: " + (num1 * num2));      // Multiplication
      System.out.println("Quotient: " + (num1 / num2));     // Division
  }
}

Variable Rules

            
    
              
  • Variable names must start with a letter, _, or $.
    • 
              
  • Cannot use reserved keywords as variable names (e.g., int class = 5; is invalid).
    • 
              
  • Variable names are case-sensitive (age and Age are different).
